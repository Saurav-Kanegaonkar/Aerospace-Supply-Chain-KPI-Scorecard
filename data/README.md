# Data Sources

Synthetic source-style data for aerospace supply chain KPI reporting.

- `entities.csv`: 36 supplier part records.
- `daily_metrics.csv`: 5,040 supplier part-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
