# Data

This folder contains synthetic aerospace supply chain reporting data generated by `scripts/score_operating_data.py`. It does not represent any real company, supplier, program, ERP system, inventory balance, or operational performance.

The structure is modeled on recurring supply chain BI sources: ERP purchase orders, ERP inventory balances, supplier and part master data, Excel shortage trackers, SharePoint quality logs, KPI documentation, refresh logs, and data-quality issue queues.

Synthetic assumptions include specialized aerospace electronics part classes, supplier tiers, long lead-time components, flight-critical and program-critical demand, late receipts, quality holds, safety stock, shortage exposure, excess inventory, and reporting refresh SLA outcomes.
